# Bank Management System - Requirements

## Version
V1.0

## Project Goal
Develop a console-based Bank Management System using Modern C++ while applying software engineering principles and object-oriented programming.

## Functional Requirements

- Create Customer
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Delete Account
- Display Active Customers
- Exit System

## Business Rules

- One customer can own only one active account.
- Deleting an account does not delete the customer.
- Every account starts with a balance of 0.
- Every financial operation requires PIN authentication.
- Customer IDs and Account Numbers must be unique.
